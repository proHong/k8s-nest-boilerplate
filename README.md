## Description

This project demonstrates the deployment of NestJS microservices using RabbitMQ for communication, orchestrated on Kubernetes.

## Getting Started

### PreDependencies

- Docker
- Kubernetes, Kubectl

### Check Dependencies

- DependenciesCheck

  ```bash
  chmod +x ./check.sh

  ./check.sh
  ```

### Running

- Run

  ```bash
  chmod +x ./run_script.sh

  ./run_script.sh
  ```

### Delete

- Delete

  ```bash
  chmod +x ./del_script.sh

  ./del_script.sh
  ```
